# 수박 게임(모작)
공주 유니티 부트캠프 페어 프로그래밍 프로젝트에 유니티에서 수박 게임의 과일 구조를 구현해보고 싶어 제작한 게임입니다.

### 1) 게임 장르
- 퍼즐 아케이드
### 2) 담당 역할
- 2인 팀) 리소스 관리 모듈과 레벨업 시스템 등의 게임 시스템 구현
### 3) 개발 기간
- 2023.10 - 2023.11 (2주일)
### 4) 개발 환경
- Windows 11, Unity Editor 2022.3.11f1, VisualStudio 2022, JsonSerialize 1.0.0, CsvHelper 30.0.1
